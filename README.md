# how to use

## get repo and run locally

### run cmd

### clone repo

```bash
type 'git clone https://github.com/michalpostek/quiz.git'
```

### install dependencies and run app

```bash
type 'cd quiz'
```

```bash
type 'npm install'
```

```bash
type 'npm run' 
```
