# how to install

### run cmd

### clone repo

```bash
type 'git clone https://github.com/michalpostek/quiz.git'
```

### go to app directory

```bash
type 'cd quiz'
```

### install dependencies

```bash
type 'npm install'
```

### run app

```bash
type 'npm run' 
```
